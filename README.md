# Online Shopper Behaviour Prediction

## Overview
This machine learning project predicts whether an online shopper will complete a purchase based on customer browsing behaviour and shopping preferences.

## Dataset
The dataset was collected using Google Forms with 166 responses. After data cleaning and duplicate removal, 164 responses were used for analysis.

## Features
- Age Group
- Shopping Frequency
- Shopping Platform
- Product Type
- Browsing Time
- Pages Viewed
- Product Reviews
- Discounts
- Free Shipping
- Trust in Ratings
- Cart Abandonment

## Machine Learning Models
- Logistic Regression
- Decision Tree Classifier

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Results
| Model | Accuracy |
|---|---|
| Logistic Regression | 94.12% |
| Decision Tree | 85.29% |

Logistic Regression achieved the highest accuracy and performed better for this binary classification problem.

## Conclusion
This project shows how machine learning can help analyze online shopping behaviour and predict purchase completion in e-commerce platforms.
